Getting Started with CDS for Code

This extension for Code allows developers to work with their solutions and data in Dynamics 365 Customer Engagement. You'll want to do these things to get started.

  • Connect to a CDS or Dynamics 365 instance
    You can connect to a CDS or Dynamics 365 instance in 3 ways - Online, On-Premises using Windows/NTLM, and Online or On-Premises using Azure Active Directory.
    1. If you've got a connection string handy, you may enter it below, otherwise, click here to open the connection window.
  • Open a workspace and start editing files
    1. Open a workspace
    2. Create a CDS or Dynamics solution if you don't already have one
    3. Export a solution to your workspace
